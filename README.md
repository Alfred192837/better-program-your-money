# Better Program Your Money 💰

微信小程序 — 个人收支记账工具。数据存储在手机本地，无需注册，即开即用。

## 功能

- 📚 **多账本**：创建多个账本（日常、旅行、项目…），分类管理
- 📝 **收支记录**：快速记录收入/支出，分类+日期+备注
- 🏷️ **常用标签**：自动记忆常用分类，一键填入
- 📊 **月度统计**：当月收支汇总 + 分类明细占比
- 🔍 **搜索筛选**：按日期范围、关键词搜索账单
- 📤 **CSV 导出**：全部数据导出为 CSV，可分享到微信
- 🗑️ **批量管理**：勾选多条记录或账本，一键删除

## 开发

微信开发者工具打开此目录即可编译运行。

## 技术

- WeChat Mini Program 原生框架
- 数据存储：`wx.setStorageSync`
- 无后端、无云服务、无需注册
